
# TypeScript API with TypeORM

## Run this project

- Install the dependencies:
```
yarn
```
- Change the ormconfig.json with your DB settings
- Run the app:
```
yarn start:dev
```
## Build and Run
- Install the dependencies:
```
yarn
```
- Change the ormconfig.json with your DB settings
- Build and run the app:
```
yarn build
yarn start:prod
```
